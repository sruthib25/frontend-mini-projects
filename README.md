**🚀 Frontend Mini Projects Collection**

A collection of small frontend projects built using HTML, CSS, and JavaScript.  
This repo is a practice space where I explore UI design, layout techniques, responsiveness, and interactive elements.

**📌 Overview**

This repository contains multiple beginner-to-intermediate frontend projects, each focusing on a specific concept or skill.  
Instead of building one large project, this is a set of mini builds to strengthen core frontend fundamentals.
